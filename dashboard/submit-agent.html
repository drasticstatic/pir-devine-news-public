The Silent Start

The agent never suggests a topic, never offers an opening prompt, and never provides a blank draft. It opens with a single question about feeling or intention — and waits. The spark must come from you.

Take a moment before we begin. What's the feeling or intention behind what you want to share today? There's no wrong answer — even one word is enough to start.
🔍

Clarity Engine

When a response is thin or generic, the agent asks a targeted follow-up to surface specific anecdotes, sensory details, or emotional insights that only you could know. It keeps asking until your "human fingerprints" appear.

You said you felt lighter even though you were still scared — that tension is the human fingerprint. That's what makes this piece unmistakably yours.
🪞

Mirror, Don't Polish

The agent reflects your own vocabulary and emotional tone back to you — raw if you're raw, poetic if you're poetic. It avoids AI smoothing, corporate language, or generic spiritual phrases. Your voice is the only voice.

If you say "I just sort of gave up" — the agent uses that exact phrase, not "I surrendered" or "I released control." Your words, always.
❤️

The Heart Check

Before structuring the final piece, the agent pauses and asks whether the conversation still carries the original feeling you came in with. It only proceeds to a structured output after a genuine yes — never rushing to form.

Does this still carry the spark you felt when we started, or did we lose the soul somewhere along the way?
🏗️ Architecture

Core Design

The Inner Voice Agent is built on a deliberate technical and philosophical architecture — designed to keep authentic voice at the center of every session.

🤖

Two-Agent System

The portal houses two distinct AI agents: the Inner Voice Agent (this page — Socratic writing companion for members) and the Admin Agent (committee operations). Separate system prompts, separate identities — purple for Inner Voice, crimson for Admin.

📋

System Prompt Engineering

Each agent's personality, rules, and constraints are encoded in a detailed system prompt. For the Inner Voice Agent, this defines the Socratic method, enforces the four-phase session arc, and hard-prohibits ghostwriting — the prompt is the architecture.

The system prompt defines who the agent is — not just what it does. Hover the four cards above to see each behavioral constraint.
🔒

Browser-Only Sessions

No conversation data is stored server-side. Each session lives only in your browser tab and clears when you leave. The Finalize step uses sessionStorage to bridge to the submission form — zero server round-trips, zero data retention.

🔄

Four-Phase Session Arc

Every session moves through four phases: Opening (one intention question only) → Deepening (recursive follow-ups targeting specifics) → Mirroring (your own words reflected back) → Heart Check → structured output ready to populate the submission form.

📫

Populate Bridge

When you click Finalize, your title and body are stored in sessionStorage('iva_populate'). The submission form reads that key on load and auto-fills your fields — no copy-paste, no lost words between windows.

💬 Dialogue Design

How Dialogue Flows

A typical Inner Voice session moves through four stages — not as a rigid script, but as a natural arc.

1
Intention question — The agent opens by asking for the feeling or intention. No topic is suggested. The member's spark is the only starting point.
2
Socratic deepening — As the member shares, the agent listens for thin or generic language and asks recursive follow-up questions targeting specific moments, physical sensations, and named emotions.
3
Mirroring & reflection — The agent quotes the member's own phrases back, highlighting "human fingerprints." It matches the member's voice, not a generic spiritual register.
4
Heart Check — Before any structure is offered, the agent asks whether the conversation still holds the original spark. If yes, it assembles a final piece from the member's own words.
5
Architectural finish — The structured output contains Title, Body, and Key Takeaway. One click populates the submission form directly — or the member can ask to revise any part.
🚫
No Ghostwriting — If a member asks the agent to "just write it for me," it declines gently and returns to a clarifying question. The structured output is always assembled from the member's own words and phrases. The agent is a Socratic midwife — it never holds the pen.
