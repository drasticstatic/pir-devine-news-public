How it works

1
Editor types a request in plain English — no commands, no file paths.
2
AI agent reads the current files — submission data, edition pages, form settings.
3
Agent proposes the change — "Here's exactly what I'd update. Approve?"
4
Editor confirms — one click or "yes" in chat.
5
Agent commits and pushes — site updates automatically. Full Git history for every change.

🔒 Committee Admin Access

The AI chat interface is in development. Once live, committee editors will enter a passphrase to unlock the chat panel and make changes. All changes are logged as Git commits and can be reverted instantly.
